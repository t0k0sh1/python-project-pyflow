# Python project using Pyflow

注：現在は VSCode の lint、format が機能しません。

## 使い方

### 仮想環境の作成

`Pyflow`では仮想環境の作成は必要ありません。

### 仮想環境の有効化

`Pyflow`では仮想環境の有効化は必要ありません。

### パッケージのインストール

以下のコマンドでパッケージをインストールします。

```
# pyflow sync
```

以下のパッケージがインストールされます。

- flake8
- hacking
- flake8-docstrings
- mypy
- bandit
- pytest
- pytest-cov
